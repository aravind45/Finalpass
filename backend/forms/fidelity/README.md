# Fidelity Forms

This directory contains PDF forms for Fidelity Investments estate settlement.

## Forms

### Estate Settlement Packet
- **File:** `fidelity-estate-settlement-packet.pdf`
- **Description:** Complete packet for settling Fidelity accounts
- **Pages:** 8
- **Fax to:** 1-800-544-0003

### Transfer Request
- **File:** `fidelity-transfer-request.pdf`
- **Description:** Request to transfer or close individual accounts
- **Pages:** 3
- **Fax to:** 1-800-544-0003

## How to Obtain Forms

1. **Call Fidelity Estate Services:** 800-544-0003
2. **Request:** "Estate Settlement Packet"
3. **Save PDFs:** Place them in this directory

## Form Field Mapping

The platform will auto-populate these fields from your estate data:
- Deceased's name
- Date of death
- Social Security Number
- Executor information
- Account numbers

## Notes

- Forms are sent via PamFax API
- Cost: ~$0.07 per page
- Delivery confirmation included
- Status tracked in database
